A complete concept-to-production course using OrCAD 17.4. Learn Schematic Capture, Footprint Creation, and PCB Layout from scratch.

Course Syllabus

1. Getting Started & Libraries
  • Intro to OrCAD 17.4 Workspace
  • Creating Projects & Title Blocks
  • Library Management: Copy, Paste, & Rename parts
  • Creating New Symbols & Design Libraries
  • Unified Search & placing components
2. Schematic Connectivity
  • Connecting wires, buses, and nets
  • Placing Net Aliases & managing groups
  • Editing component values & properties
  • Reference Designator updates (Annotation)
  • Defining Differential Pairs (Manual/Auto)
3. Validation & BOM
  • Running Design Rule Check (DRC)
  • Correcting electrical violations
  • Custom Bill of Materials (BOM) in Excel
  • Smart PDF Generation
  • Netlist generation for PCB Editor
4. Footprint Creation
  • Through-Hole Footprint creation
  • Surface Mount (SMD) Footprints
  • COMBI SMT-TH Footprints
  • BGA / PGA complex footprint creation
  • Padstack creation fundamentals
5. PCB Setup & Placement
  • Board Stack-up & Layer setup
  • Mechanical symbols & Mounting holes
  • Component Placement & Cross-probing
  • Unplaced component reports
  • Board shape creation
6. Routing & Planes
  • Setting up Constraint Sets (Physical/Electrical)
  • Differential Pair rules & routing
  • Interactive Routing & Trace cleanup
  • Creating Copper Shapes (Power/Ground Planes)
  • Merging geometric shapes
7. DRC & Final Polish
  • Running PCB Design Rule Check
  • Identifying & Fixing DRC errors
  • Adding Dimensions & Fab notes
  • Drill Chart & Cross-section Chart creation
  • Silkscreen adjustments
8. Manufacturing & 3D
  • Gerber Artwork Generation (IPC-2581)
  • NC Drill File generation
  • STEP Mapping for 3D Models
  • 3D Collision Detection & Resolution
  • Final Archive & Release

What You Will Master

By the end of this course, you will be able to work professionally:

Basic Electronics Theory & Circuit Design
